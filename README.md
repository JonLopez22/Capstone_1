Capstone Project: Business Analysis using SQL and Excel

Project Description:
This project analyzes sales data for EporiUm, a student bookstore that sells textbooks, technology accesories, art supplies, and more using both in-store and online.

Sales Territory:
- The focus of anaylsis is on Colorado's in-store sales, managed by sales manager Jim Heck under the West region. The regional director is Cassie Chambers.
- The West region includes 18 stores across Colorado that include Aurora, Boulder, Denver, Fort Collins, Greeley and more.

What this project covers:
- Total revenue and data range for the Colorado territory
- Month by month revenue breakdown
- Comparison of Colorado in-store sales vs West Region online sales
- Number of transactions and average transaction size by product category
- Store peformance rankings across all 18 Colorado locations
- Recommendation for where to focus sales attention for the next quarter

Excel PivotCharts:
- Two of the SQL query results have been exported to CSV and visualized in Excel using PivotCharts
    1. "Month by Month Revenue Breakdown" - illustrates how Colorado's in-store revenue trended over time from 2022 through 2025.
    2. "Store Peformance Rankings" = visualizes total revenue by store location, highlighting top and bottom peformers across the 18 Colorado stores.

 Tools used:
 - MySQL, for data querying and analysis
 - Excel, for data visualization via PivotCharts
 - Gitbash/Github, for project publishing
